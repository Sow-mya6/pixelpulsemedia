# 🚀 PixelPulse Media

A full-stack digital marketing agency website built using the MERN stack. The application provides a modern and responsive platform for showcasing services, portfolio, company information, and managing content through an admin dashboard.

## ✨ Features

- Responsive user interface
- Home, About, Services, Portfolio, and Contact pages
- Admin Dashboard
- REST API integration
- MongoDB database connectivity

## 🛠️ Tech Stack

**Frontend**
- React.js
- Vite
- JavaScript
- CSS3

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

## 📁 Project Structure

pixelpulse-media/

## ▶️ Running the Project

### Start the Backend

```bash
cd server
npm install
npm start
```

### Start the Frontend

```bash
cd client
npm install
npm run dev
```
